Bottom Navigation

Navegación principal. Fija en la parte inferior. 3–5 ítems. El pulgar alcanza fácilmente. Reemplaza el sidebar en mobile.

Contenido de la pantalla

.bottom-nav .bottom-nav-item.active ← línea superior Terracota .bottom-nav-item <i data-lucide="icon-name"></i> ← Lucide icon 22px .bottom-nav-badge ← número de notificaciones "Label" /* padding-bottom: env(safe-area-inset-bottom) — iPhone notch */

Reglas de uso

✓ Máximo 5 ítems — más de 5 confunde al usuario

✓ Íconos + texto — texto siempre visible, nunca solo ícono

✓ Badge para notificaciones — número máximo "9+"

✗ No usar para acciones — solo para navegación entre secciones principales

✗ No mezclar con sidebar lateral — elige uno según el dispositivo
